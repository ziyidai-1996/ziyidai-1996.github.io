# Ziyi Dai Homepage

1. 打开 github desktop (紫色软件)
2. Fetch Origin 从网上拉取更新
3. Open Explorer 打开对应文件夹
4. _pages/includes/ 文件夹中的每个.md文件代表网页上的某一块
5. 修改对应的块并 **保存**
6. 打开GitHub Desktop，在左侧Description上面的小输入框中随便输入一些内容
7. Commit to main 点击
8. Push Origin 点击
9. 等待网页自动更新
